# Как добавлять новые работы

Сайт сейчас статический, без отдельной админки. Новые рисунки добавляются через файлы проекта, но схема максимально простая.

## Куда класть изображения

Все новые работы складывайте в папку `фото/`.

Лучше использовать:

- `.jpg` или `.png`
- вертикальные фото хорошего качества
- понятные имена файлов, например `kot-portret-01.jpg`

## Что менять в коде

Откройте `script.js` и найдите массив `works`.

Добавьте в него еще один объект по образцу:

```js
{
  title: "Название работы",
  meta: "краткое описание техники",
  image: "фото/kot-portret-01.jpg",
  alt: "Краткое описание рисунка для сайта"
}
```

## Что значат поля

- `title` - короткое название работы
- `meta` - подпись под названием
- `image` - путь к файлу изображения
- `alt` - текст для доступности и поисковиков

## После добавления

1. Откройте `index.html`.
2. Проверьте, что работа появилась в горизонтальной галерее.
3. Нажмите на нее и убедитесь, что она открывается крупно.

## Если нужен доступ без кода

Для загрузки работ без редактирования файлов лучше следующим этапом перевести сайт на CMS. Самый понятный вариант для такого проекта: `Next.js + Sanity + Vercel`.
