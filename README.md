# Лендинг Ани

Статический лендинг-портфолио для художницы `@narisuyu_o`.

## Как открыть

Откройте файл `index.html` в браузере. Сервер не нужен.

## Как добавить работу в каталог

1. Положите новое изображение в эту же папку.
2. Откройте `script.js`.
3. Добавьте объект в массив `works`:

```js
{
  title: "Название работы",
  meta: "цветной карандаш · по фото",
  category: "pets",
  image: "new-image.jpg",
  alt: "Краткое описание изображения"
}
```

Доступные категории: `pets`, `people`, `process`.

Контакт на сайте сейчас только Telegram: `https://t.me/narisuyu_o`.
