# Лендинг Ани

Статический лендинг-портфолио для художницы `@narisuyu_o`.

## Как открыть

Откройте `index.html` в браузере. Сервер не нужен.

## Как обновлять работы

Подробная инструкция лежит в файле `ДОБАВЛЕНИЕ-РАБОТ.md`.

Коротко:

1. Положите новое изображение в папку `фото/`.
2. Откройте `script.js`.
3. Добавьте новую работу в массив `works`.

Пример:

```js
{
  title: "Название работы",
  meta: "краткое описание техники",
  image: "фото/имя-файла.jpg",
  alt: "Описание изображения для сайта"
}
```

Контакт на сайте сейчас только Telegram: `https://t.me/narisuyu_o`.
