Простая админка

Добавление работ без правки кода

Здесь можно загрузить новое изображение, добавить название и описание, а затем отправить все прямо в GitHub. После публикации Vercel подтянет новую версию сайта из ветки main.

Что понадобится

  • GitHub fine-grained token с доступом к этому репозиторию
  • готовое изображение работы в JPG, PNG или WebP
  • короткое название и подпись для карточки

Шаг 1

Доступ к GitHub

Токен сохраняется только в браузере на этом устройстве. В код сайта он не попадает.

Шаг 2

Как проходит публикация

  1. Выбери изображение и заполни карточку работы.
  2. Нажми кнопку публикации внизу формы.
  3. Админка добавит файл в папку фото/ и обновит works.json.
  4. Через пару минут новая работа появится на основном сайте.

Лучше использовать токен от аккаунта, который владеет этим GitHub-репозиторием и связан с Vercel. Так автодеплой проходит ровнее.

Шаг 3

Новая работа

После публикации админка сразу покажет результат и ссылку на коммит.
