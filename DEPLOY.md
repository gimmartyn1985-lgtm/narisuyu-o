# Как запустить сайт, чтобы Аня могла пользоваться сама

Есть два разных уровня запуска.

## Вариант 1. Быстро выложить сайт на Vercel

Это самый быстрый вариант: сайт будет доступен по ссылке, но новые работы придется добавлять через файлы проекта.

### Что нужно

- аккаунт GitHub
- аккаунт Vercel
- эта папка с сайтом

### Шаги

1. Создайте новый репозиторий на GitHub, например `anya-artist-landing`.
2. Загрузите туда все файлы из этой папки.
3. Откройте Vercel: `https://vercel.com/new`.
4. Нажмите `Import Git Repository`.
5. Выберите репозиторий `anya-artist-landing`.
6. Framework Preset оставьте `Other`.
7. Build Command оставьте пустым.
8. Output Directory оставьте пустым.
9. Нажмите `Deploy`.

После этого Vercel даст ссылку вида:

```text
https://anya-artist-landing.vercel.app
```

## Вариант 2. Чтобы Аня сама добавляла работы через админку

Для самостоятельного обновления работ лучше не оставлять сайт просто статическим HTML. Нужна CMS: личная админка, где можно загрузить фото, название, категорию и описание.

Рекомендованная связка:

- Vercel — хостинг сайта
- Sanity — админка для работ и текстов
- GitHub — хранение кода

Как это будет выглядеть для Ани:

1. Она заходит в админку Sanity.
2. Нажимает `New artwork`.
3. Загружает фото.
4. Пишет название, категорию и описание.
5. Нажимает `Publish`.
6. Работа появляется на сайте.

Для этого текущий статический сайт нужно перевести в Next.js + Sanity.

## Почему не просто Vercel

Vercel отлично публикует сайт, но сам по себе не является удобной админкой для художницы. Если оставить только Vercel, обновление работ будет через код, GitHub или ручную замену файлов.

## Лучший практический план

1. Сейчас выложить текущую версию на Vercel как MVP.
2. После утверждения дизайна перевести сайт на Next.js + Sanity.
3. Дать Ане доступ только к Sanity, а не к коду.
4. Подключить домен, например `narisuyu-o.ru` или похожий.

## Что уже подготовлено

- `index.html` — страница сайта
- `styles.css` — стили
- `script.js` — каталог и фильтры
- `vercel.json` — базовый конфиг для Vercel
- `README.md` — как добавлять работы в текущую файловую версию
